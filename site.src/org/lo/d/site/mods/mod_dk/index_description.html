門番メイドのお仕事

門番モードにする

かをCommonsの機能によって追加される「書ける本での指示」の「モード変更」によって門番モードに変更します。
モード名は「DoorKeeper」もしくは「dk」となります。 書ける本に書く指示としては、
m,dk;
のようになります。

自動ドア(人力)モード

追従モードの時、ドアを自動的に開け閉めしてくれます。
プレイヤーが見ている、プレイヤーからの距離が6m以内のドアを開けに行きます。
プレイヤーが見ておらず、プレイヤーからの距離が7m以上離れたドアを閉めに行きます。
それぞれ一定距離まで近づく必要があり、近づけない場合は動きません。

お出迎えモード

Freedomモードの時、プレイヤーを探知して設置されたレバーを片っ端からON/OFFしていきます。
メイドさんがプレイヤーを見ることが出来、プレイヤーが8mより近くにいる時、
近くにあるレバーを片っ端からONにしていきます。
メイドさんがプレイヤーを見失って一定時間が経過するか、プレイヤーが10m以上離れると、
近くにあるレバーを片っ端からOFFにしていきます。
あまり高いところには手が届きません。
プレイヤーを見つけた時にちょっと喜びます。
