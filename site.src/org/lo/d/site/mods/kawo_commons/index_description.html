インストール

ダウンロードしたファイルをmodsに置いて下さい。
初期状態では、「アンインストール状態」になっていますので、
コンフィグを開き「enableUninstallMode=false」にしてModを有効化して下さい。

！注意！アンインストール

当MODをアンインストールする際、必ずコンフィグを開き「enableUninstallMode=true」にして、
「アンインストール状態」にしてから一度Minecraftを起動し、その後Minecraftを終了させてから、
MODを削除して下さい。
上記手順を行わずにMODを削除してワールドを読み込んだ場合、全てのスポーン済みのメイドさんが消滅します。

使い方

書ける本による指示

書ける本に書式に従って各種指示を書き、その本を持った状態でメイドさんを右クリックすると
様々な指示を与えることができます。

書式

[コマンド],[引数],[引数];
複数行に渡って記載できます。
コマンドの大文字小文字は無視されます。
「;」は単一の指示を記載する場合、省略可能です。

モード変更指示

mode,[モード名]
もしくは
m,[モード名]

契約中で、待機中"でない"メイドさんのモードを[モード名]で指定されたモードに変更します。
各モード割り当てられた[モード名]は以下のとおりです。
従者：「Escorter」もしくは「Es」
フェンサー：「Fencer」もしくは「Fe」
アーチャー：「Archer」もしくは「Ar」
お料理：「Cooking」もしくは「Co」
ヒーラー：「Healer」もしくは「He」
薬剤師：「Pharmacist」もしくは「Ph」
毛刈り：「Ripper」もしくは「Ri」
トーチャー：「Torcher」もしくは「To」

他にもあるかも…？

砂糖カウント

メイドさんにカーソルを合わせると現在所持している砂糖の量を表示します。
非表示にしたい場合はコンフィグからOFFに出来ます。(SugarCountVisible=false)

拡張API(開発者向け)

追加したモードの専用GUIを開く、interactを完全に上書きする、などなど。

ニーズがありそうなら解説を書いていきます。
